# 6-7 Daily

Further adventures in Vibe Coding.

A daily trivia game where the goal is to hit the "Sweet Spot".

## How to Play

Every day, you get 3 rounds of top-10 list categories (e.g., "Largest Countries", "Best Selling Albums").
Your goal is to guess an item that ranks **6th** or **7th** on the list.

### Scoring
The aim is to get the **lowest score possible**.

- **Rank 6 or 7**: 0 Points (Perfect!)
- **Rank 1**: +5 Points
- **Rank 2**: +4 Points
- **Rank 3**: +3 Points
- **Rank 4**: +2 Points
- **Rank 5**: +1 Point
- **Rank 8-10 or Miss**: +10 Points

## Tech Stack

- **HTML5**
- **React** (via CDN)
- **Tailwind CSS** (via CDN)
- **Babel** (via CDN for JSX compilation)

## How to Run

Simply open `index.html` in any modern web browser. No build step required!
